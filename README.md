# Implementación del algoritmo cuántico de Bernstein Vazirani
Entregable Universidad de la Laguna: Introducción a la Computación Cuántica

En este repositorio se encuentra un notebook de jupiter en python, que utilizando la librería de computación cuántica de Qiskit ejecuta el algoritmo de Bernstein Vazirani.

El entorno donde se ha ejecuta es en Colab Enterprise de Vertex AI en Google Cloud.
